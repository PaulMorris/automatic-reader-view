Reader View Websites

Firefox will attempt to show pages on these websites in Reader View. Enter a list of website addresses (URLs) separated by commas. For example: "www.mozilla.org, www.ubuntu.com, wordpress.org" Include any subdomains like "www." Target just a subsection of a website like this: "wordpress.org/news"

Try to Show All Websites in Reader View

Attempt to show all pages in Reader View by default. Note that Firefox will only show certain pages in Reader View in any case, even if this box is checked.

Not Reader View Websites

When "Try to Show All Websites in Reader View" is in effect, these sites will not be (automatically) shown in reader view.
